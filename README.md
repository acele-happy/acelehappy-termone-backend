# acelehappy-termone-backend
### Steps to run the app.

#### Prerequisites
- ```Node.js```
- ```Npm or Yarn```

<br>

Clone the repo
```bash
git clone https://github.com/acele-happy/acelehappy-termone-backend
```

Install Dependencies
```
npm i
```
or
```
yarn
```

Run the server
```
npm start
```
or
```
yarn dev
```

Run tests
```
npm  test
```
or
yarn test
```
